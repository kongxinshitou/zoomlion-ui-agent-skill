# Homepage Refactor Prompt

Use `zoomlion-ui` to refactor the current homepage into a Zoomlion-inspired industrial enterprise homepage.

## Goal

Create a professional, product-first, high-end manufacturing visual direction while preserving all existing business logic.

## Constraints

- Do not change routes, API calls, data fetching, state shape, permissions, analytics, or component public props.
- Keep existing real content where available.
- Use placeholder imagery only when no authorized product/factory/service assets exist, and label it as replaceable.
- Use green as a precise accent, not a full-page flood.
- Avoid generic SaaS blue-purple gradients, cute illustrations, cyberpunk neon, and luxury black-gold styling.

## Expected structure

1. Enterprise header/navigation
2. Hero banner with machinery/manufacturing/global service imagery
3. Product category section
4. Technology / intelligent manufacturing section
5. Data achievements module
6. News center
7. Service/global network module
8. ESG / green development module
9. Dark industrial footer

## Required output

After editing, report:

- Files changed
- Brand rules applied
- Logic preserved
- Checklist result using `references/zoomlion-ui-checklist.md`
- Validation commands run or not run
